# BFF
フロントエンドのための処理が詰まってるサーバーのことを BFF と呼ぶという取り決めみたいなものである。
普通の web api と BFF の垣根ってどこ？と言われたら結構曖昧なので答えにくい。
なんか、フロントエンド用に特化したなんらかの処理をしてるバックエンドのサービスなんだなあくらいの理解で良い。
例えば SSR を行うサーバーも BFF と呼んでも差し支えない。
